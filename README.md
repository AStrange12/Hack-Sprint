# HealthPredict AI - Multimodal Clinical Deterioration Prediction

This is a clinical decision support system designed to predict patient outcomes using time-series vitals and AI-driven clinical reasoning.

## 🚀 Quick Start
1. **Login**: Use the "Doctor Portal" to manage individual patients.
2. **Admin**: Use the "Admin Portal" for bulk Excel imports and OCR scanning.
3. **Analyze**: Go to a patient's profile to run ML-driven or AI-driven predictions.

## 🛠 Features
- **Dual Prediction Engine**: Dataset-driven (k-NN) + AI-driven (Genkit/Gemini).
- **Bulk Import**: Excel and CSV parsing.
- **OCR Scanner**: Extract data from medical reports.
- **Real-time Trends**: Visualize vital signs over time with Recharts.
